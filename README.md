# Vehicle Routing Program

## Build binary
```bash
go build
```

## Usage in isolation

```bash
./routing_solver <input_file>
```

## Usage alongside test cases
```bash
python3 evaluateShared.py --problemDir test_problems --cmd "./routing_solver"
```
